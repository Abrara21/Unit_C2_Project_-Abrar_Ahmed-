Tables

The Essential Elements for Tables

HTML tables are created using the following essential elements:

  • <table>: Creates the table.
  • <tr>: Creates a row.
  • <th>: Creates a heading cell.
  • <td>: Creates a data cell.

Example Table

Below is an example demonstrating the use of these elements:

Item Quantity Price
Apples 5 $3.00
Bananas 8 $4.00
Oranges 3 $2.50

The border="1" attribute is used to add a simple border around the table, and style="border-collapse: collapse;" makes the table cleaner.
