HTML Formatting Tags

Common HTML Formatting Tags

HTML provides various tags to format text. Below are some examples:

  • <strong>: Makes text bold.
  • <em>: Emphasizes text (usually italic).
  • <small>: Displays smaller text.
  • <mark>: Highlights text.
  • <del>: Displays strikethrough text.
  • <ins>: Indicates inserted text (underlined).
